## 步驟

### 第一步: 下載
1. 請按右上綠色按鍵"Clone or download"
2. 按 "Download ZIP"

### 第二步: 取代原本資料夾
3. 將下載後的檔案解壓縮
4. 把DICTS裡面所有東西複製到你本機上的這個路徑下: c:\windows\IME\IMETC\DICTS\
5. 把zh-tw裡面的所有東西複製到你本機上的這個路徑下: c:\windows\system32\zh-tw\
